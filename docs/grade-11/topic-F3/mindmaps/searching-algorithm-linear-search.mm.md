# Tìm kiếm

## Bài toán

- Đầu vào: mảng `A` và giá trị cần tìm `k`
- Đầu ra: chỉ số tìm thấy `k` hoặc `-1`

## Thuật toán

### Tìm kiếm tuần tự

#### Ý tưởng
    
- Duyệt mảng từ đầu đến cuối
- So sánh từng phần tử `A[i]` với `k`

#### Viết chương trình

- Dùng vòng lặp `for`
