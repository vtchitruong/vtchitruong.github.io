## Câu lệnh SELECT

### Dùng để truy hồi dữ liệu

### Cú pháp cơ bản: `select ... from ... where ...`

- `select`: chọn các cột cần hiển thị
- `from`: từ bảng (mà) có chứa các cột liên quan
- `where`: điều kiện lọc dữ liệu
- `order by`: sắp xếp kết quả hiển thị
