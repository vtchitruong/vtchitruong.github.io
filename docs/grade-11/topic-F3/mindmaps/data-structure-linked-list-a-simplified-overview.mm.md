# Danh sách liên kết

## Về cấu trúc

- Gồm các phần tử liên kết với nhau thông qua tham chiếu

## Phần tử

- Mỗi phần tử gọi là node
- Gồm hai thành phần:

  - Dữ liệu `data`: lưu dữ liệu
  - Tham chiếu `next`: lưu địa chỉ node kế tiếp

- Node cuối cùng trỏ đến `None`

## Lợi ích

- Thêm, xoá phần tử dễ dàng, không cần di chuyển.
