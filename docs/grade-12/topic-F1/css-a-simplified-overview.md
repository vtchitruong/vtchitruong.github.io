# Khái quát về CSS

!!! abstract "Tóm lược nội dung"

    Bài này trình bày khái quát về CSS.

## Định nghĩa

!!! note "CSS"

    Là **ngôn ngữ định kiểu** dùng để mô tả **cách thể hiện của các phần tử** trong tài liệu HTML.

Cụ thể, CSS mô tả các yếu tố trực quan bao gồm: bố cục, màu sắc, font chữ và các hiệu ứng chuyển động.

---

## Mục đích

Sử dụng CSS nhằm thực hiện nguyên lý **tách biệt nội dung và hình thức** của trang web:

- HTML: đóng vai trò **bộ khung**, kiểm soát cấu trúc và nội dung.
- CSS: đóng vai trò **lớp áo**, kiểm soát hình thức, tính thẩm mỹ và trải nghiệm thị giác.

---

## Lợi ích

1. **Khả năng tùy biến**

    - CSS có thể đáp ứng những yêu cầu phức tạp về hình thức, bố cục và hiệu ứng nhờ có các công cụ hiện đại như Grid hoặc Flexbox.
    - CSS giúp bố cục trang web tự động tương thích với nhiều loại thiết bị khác nhau nhờ có 
    Media Queries.

2. **Nhất quán về hình thức**

    CSS bảo đảm sự nhất quán và đồng bộ về hình thức cho các trang web trong cùng một website.

3. **Thuận tiện trong bảo trì và nâng cấp**

    Khi cần thay đổi hoặc nâng cấp, ta chỉ cần chỉnh sửa một số chỗ, thường là các biến CSS, thay vì phải sửa từng tập tin HTML riêng lẻ hay làm ảnh hưởng nội dung hiện có.

4. **Cải thiện hiệu suất tải trang**

    - CSS giúp mà nguồn HTML ngắn gọn hơn.
    - Trình duyệt có thể lưu tập tin CSS vào bộ nhớ đệm, giúp giảm băng thông và tăng tốc độ hiển thị cho những lần truy cập sau.
