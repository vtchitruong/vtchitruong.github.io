# Truy vấn dữ liệu

## `WHERE`

### Điều kiện của dữ liệu cần trích xuất

- Các toán tử so sánh dữ liệu kiểu số
- Toán tử `LIKE`: so sánh gần bằng
- Toán tử logic `AND`, `OR`, `NOT`  

## `ORDER BY`

### Sắp xếp dữ liệu trả về

- `ASC`: tăng dần
- `DESC`: giảm dần
