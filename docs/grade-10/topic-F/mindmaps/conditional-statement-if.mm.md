# Cấu trúc điều kiện

## Dùng để giải quyết bài toán có nhiều trường hợp.

## if thiếu

- `if ...`
  - Nếu đúng thì thực thi, không thì thôi.

## if đủ

- `if ... else ...`
  - Dành cho bài toán có hai trường hợp.

- `if ... elif ... else ...`
  - Dành cho bài toán từ ba trường hợp trở lên.
