# Xây dựng cơ sở dữ liệu

## Tạo bảng

- Cú pháp SQL `CREATE TABLE`
- Có thể tạo khoá chính và khoá ngoại ngay trong câu lệnh `CREATE TABLE`

## Xem lược đồ quan hệ

- Chọn menu **ERD For Database**

## Thêm mẫu tin vào bảng

- Nhập cho bảng cha trước, rồi mới nhập bảng con.
- Cú pháp SQL `INSERT INTO ... VALUES`
