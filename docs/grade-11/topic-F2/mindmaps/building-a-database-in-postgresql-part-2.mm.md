# Xây dựng cơ sở dữ liệu

## Tạo bảng

### Cú pháp SQL `CREATE TABLE`

### Kiểu dữ liệu

- `char`
- `varchar`
- `date`
- `smallint`

### Khai báo khoá chính

- `PRIMARY KEY`

## Thêm mẫu tin vào bảng

- Cú pháp SQL `INSERT INTO ... VALUES`

## Xem dữ liệu trong bảng

- **View/Edit Data** > **All Rows**
