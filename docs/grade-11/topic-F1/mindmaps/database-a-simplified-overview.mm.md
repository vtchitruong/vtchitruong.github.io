# Hệ cơ sở dữ liệu

## Cơ sở dữ liệu

- Là tập hợp dữ liệu.
- Tính chất

    - Tính cấu trúc
    - Tính nhất quán
    - Tính toàn vẹn
    - Tính an toàn

## Hệ quản trị cở sở dữ liệu

- Là phần mềm
- Chức năng:
    - Lưu trữ
    - Quản lý
    - Xử lý
    - Truy vấn

- Ví dụ: Oracle, MySQL, Microsoft SQL Server, PostgreSQL, MongoDB

## Phần mềm ứng dụng

- Giúp người dùng thuận tiện hơn khi thao tác với cơ sở dữ liệu.
